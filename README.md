# 💰 Expense Tracker Application

## 📌 Overview
The Expense Tracker is a Python-based application that helps users manage their daily expenses efficiently. It allows users to record, view, and track spending, making it easier to monitor financial habits.

This project was developed as part of my internship to improve Python programming and problem-solving skills.

---

## 🚀 Features
- Add new expenses with amount and category
- View all recorded expenses
- Calculate total expenses
- Categorize spending (e.g., Food, Travel, Bills)
- Simple and user-friendly interface
